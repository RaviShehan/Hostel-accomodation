# Smart Hostel and Boarding Management Platform

Complete student accommodation administration platform with:

- Administrator and warden web dashboard
- Installable student mobile PWA
- Node.js REST API
- Authentication and role-based access control
- Room, resident, complaint, visitor, payment, notice, and notification workflows
- Persistent local demonstration data
- Architecture, ERD, API, workflow, and demo documentation
- Automated end-to-end API tests

The runnable project and complete instructions are in [`outputs/README.md`](outputs/README.md).

## Quick start

```powershell
cd outputs
node server.js
```

Open `http://localhost:4173`.
